# 🌸 Recipe Manager
 
> Консольное приложение для управления рецептами: создавайте блюда, добавляйте ингредиенты, масштабируйте порции и формируйте список покупок.
 
---
 
## 📋 Описание
 
Проект реализует систему управления рецептами на Python с использованием объектно-ориентированного программирования. Приложение позволяет:
 
- создавать ингредиенты с указанием количества и единиц измерения
- составлять рецепты и добавлять в них ингредиенты
- масштабировать рецепты под нужное количество порций
- вести список покупок на основе нескольких рецептов
- работать с диетическими рецептами (веган, без глютена, кето и др.)
---

---
 
## 🌷 Классы проекта
 
| Класс | Описание |
|---|---|
| `Ingredient` | Хранит название, количество и единицу измерения продукта |
| `Recipe` | Рецепт блюда с набором ингредиентов и методом масштабирования |
| `DietaryRecipe` | Наследник `Recipe` с указанием диетической категории |
| `ShoppingList` | Список покупок из нескольких рецептов с суммированием одинаковых ингредиентов |
